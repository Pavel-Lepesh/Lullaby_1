Перед запуском парсера нужно импортировать следующие пакеты в терминале:
pip install beautifulsoup4
pip install requests

В строках 19 и 58 можно выставить путь до папки, в которую будут загружены
файлы mp3(заранее создайте папку).

Аналагично в строке 80 можно установить путь для csv файла вместо "list_of_songs.csv",
В конце пути обязательно указывайте название файла с расширением - этот файл python 
создаст сам, вам не нужно его заранее создавать.

Если на сайте изменится количество страниц, либо вы захотите скачать файлы с конкретных страниц,
то в строке 77 можно изменить параметры функции range(1, 39), первый аргумент - это номер первой страницы,
второй - номер последней страницы + 1.

Запуск кода осуществляется посредством ресурса в котором вы его запускаете.
